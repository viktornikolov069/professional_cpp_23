## ‚öôÔ∏è Building & Running
Most examples can be compiled with a C++20 (or later) compiler.  

Example using `g++`:
```bash
g++ -std=c++20 -Wall -Wextra -o example examples/example.cpp
./example
```

Or, if using CMake:
```bash
mkdir build && cd build
cmake ..
make
```
